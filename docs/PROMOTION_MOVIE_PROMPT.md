## Visual Design Guidelines for Veo3

### Overall Visual Style
**Style**: Minimal, clean, contemporary
**Aesthetic**: Simplified visual elements with focus on essential information
**Approach**: Less is more philosophy throughout all scenes

### Color Palette Specification
**Primary Colors**:
- White (#FFFFFF) - Main background
- Soft Blue (#E3F2FD) - AI technology accent
- Light Gray (#F5F5F5) - Secondary backgrounds

**Accent Colors**:
- Gentle Pink (#FCE4EC) - Personal color representation
- Warm Beige (#FFF8E1) - Natural skin tone complement

**Usage Guidelines**:
- Maximum 3 colors per scene
- White dominates as primary background (70%+)
- Accent colors used sparingly for key elements only
- No gradients or complex color transitions

### UI/UX Animation Style
**Animation Type**: Simple, linear animations with minimal keyframes
**Line Art**: Clean line illustrations with minimal stroke count
**Elements**:
- Simple geometric shapes
- Minimal iconography
- Clean typography without decorative elements
- Subtle fade-in/fade-out transitions only

### Technical Animation Specifications
**Motion**:
- Linear easing (no complex curves)
- Maximum 2-3 moving elements per scene
- Static backgrounds preferred
- Minimal particle effects (if any)

**Interface Elements**:
- Flat design principles
- No shadows or 3D effects
- Simple button designs
- Clean app interface mockups

### Scene-Specific Visual Instructions
All scenes should incorporate:
- **Minimalist backgrounds** with plenty of white space
- **Simple character designs** with clean lines
- **Limited color applications** following the restricted palette
- **Subtle animations** that don't distract from the core message
- **Clean typography** with ample spacing

### Veo3 Style Keywords
Include these keywords in all prompts:
- "minimalist style"
- "clean line art"
- "limited color palette"
- "simple animation"
- "flat design"
- "white background dominant"
- "subtle transitions"
- "geometric simplicity"
