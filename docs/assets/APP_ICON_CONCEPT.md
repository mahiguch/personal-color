# AIスタイリスト - アイコンデザインコンセプト

## 🎨 デザインコンセプト

### コア要素
- **鏡/レンズ**: 自分を見つめる・診断のシンボル
- **カラーパレット**: パーソナルカラーの多様性を表現
- **AI/星**: 技術的な診断を表現する輝き
- **子ども向け**: 親しみやすく温かみのあるデザイン

### カラーパレット
- **メインカラー**: パステルピンク (#FFB6C1) - 温かさと親しみやすさ
- **アクセント**: ソフトブルー (#87CEEB) - 冷静さとAI技術
- **ベース**: ピュアホワイト (#FFFFFF) - 清潔感と安全性
- **グラデーション**: パステルレインボー - カラーバリエーション

### デザイン構造
```
中央: 鏡形状のフレーム（ピンクからブルーのグラデーション）
内側: キラキラとした星/光の効果
周囲: 4色のカラースウォッチ（イエベ・ブルベを表現）
背景: 柔らかなグラデーション
```

### 技術要件
- サイズ: 1024×1024px（App Store用）
- フォーマット: PNG（透明背景なし）
- 角丸処理: なし（iOSが自動適用）
- 視認性: 小サイズ（20×20px）でも識別可能

## 🖼️ ビジュアル構成

### レイアウト
1. **背景（100%）**: 柔らかなグラデーション（ピンク→ホワイト→ブルー）
2. **メインオブジェクト（70%）**: 円形の鏡/レンズ
3. **カラースウォッチ（各15%）**: 4方向に配置された色見本
4. **エフェクト（20%）**: 星・キラキラ効果

### 色彩設計
- **イエローベース系**: 温かみのある黄色・オレンジ・コーラル
- **ブルーベース系**: 爽やかな青・紫・ローズピンク
- **グラデーション**: 自然な色彩遷移
- **コントラスト**: 各サイズで視認性確保
