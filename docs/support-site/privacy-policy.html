🔒 プライバシーポリシー

最終更新日：2025年8月22日

1. 基本方針

パーソナルカラー診断アプリ（以下「当アプリ」）は、ユーザーのプライバシー保護を最重要事項として取り組みます。特に、小学生を含む子どもたちが安心して利用できるよう、厳格なプライバシー保護措置を講じています。

2. 収集する情報

2.1 撮影画像

  • 収集目的： パーソナルカラー診断のためのAI画像解析
  • 収集方法： アプリ内のカメラ機能による顔写真の撮影
  • 処理内容：
    • 肌色の色相・彩度・明度の分析
    • イエローベース・ブルーベースの判定
    • 診断理由の生成
  • 保存期間： 診断完了後に即座に削除（サーバー上に保存されません）
  • 第三者提供： 一切行いません

2.2 技術的情報

  • アプリの使用状況： クラッシュレポートやエラーログ（個人を特定する情報は含まれません）
  • 診断実行回数： 統計目的のみ（個人との紐付けは行いません）
  • デバイス情報： アプリの動作改善のための基本的な端末情報

2.3 収集しない情報

  • 個人識別情報： 氏名、住所、電話番号等は一切収集しません
  • 位置情報： GPS等の位置情報は収集しません
  • 連絡先： アドレス帳等の情報は収集しません

3. 情報の利用目的

  • パーソナルカラー診断サービスの提供
  • 診断結果の表示と説明の生成
  • アプリの動作改善・バグ修正
  • サービス利用状況の把握（個人特定不可能な形式）

4. データの管理・セキュリティ

4.1 撮影画像の取り扱い

1. 撮影 → 2. サーバー送信 → 3. AI解析 → 4. 結果返却 → 5. 即座に削除
  • 暗号化通信： HTTPS/TLSによる通信の暗号化
  • 一時処理： サーバー上での一時的な処理のみ
  • 自動削除： 診断完了と同時に自動削除
  • バックアップなし： 画像のバックアップは作成されません

5. 第三者への提供

撮影画像および個人に関する情報は、以下の場合を除き第三者に提供しません：

  • 法令に基づく開示要求がある場合
  • ユーザーの生命、身体、財産の保護に必要な場合
  • 公衆衛生の向上に必要な場合

6. 子どものプライバシー保護

6.1 特別な配慮

  • 13歳未満： より厳格なプライバシー保護措置
  • 保護者向け： 保護者の方への透明な情報提供
  • 安全設計： 子どもが安心して使える機能設計

6.2 保護者の方へのお願い

  • お子様のアプリ使用状況をご確認ください
  • プライバシーに関してご質問がある場合は、お気軽にお問い合わせください
  • 不適切な使用を発見された場合は、直ちにご連絡ください

7. 国際的な取り扱い

7.1 適用法令

  • 個人情報保護法（日本）
  • GDPR（EU一般データ保護規則）- 該当する場合
  • COPPA（米国児童オンラインプライバシー保護法）- 子どものプライバシー保護
