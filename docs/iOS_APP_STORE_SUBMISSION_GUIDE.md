## 📋 Step 1: App Store Connect アプリ作成

### 1.1 Apple Developer Program確認

```bash
# Apple Developer Account の確認
open https://developer.apple.com/account/
```

**必要条件:**
- ✅ Apple Developer Program 加入済み（年会費$99）
- ✅ iOS Development Certificate 作成済み
- ✅ App ID 登録済み（`com.personal-color.diagnosis-app`）
- ✅ Provisioning Profile 作成済み

### 1.2 App Store Connect でアプリ作成

```bash
# App Store Connect にアクセス
open https://appstoreconnect.apple.com/
```

**作成手順:**

1. **「マイApp」→ 「+」ボタンをクリック**
2. **「新規App」を選択**
3. **基本情報を入力:**
   - **プラットフォーム**: iOS
   - **名前**: パーソナルカラー診断アプリ
   - **プライマリ言語**: 日本語
   - **バンドルID**: `com.personal-color.diagnosis-app`
   - **SKU**: `personal-color-diagnosis-001`（任意のユニークID）

4. **「作成」をクリック**

### 1.3 App Store Connect 基本設定

**アプリ情報の設定:**
- **カテゴリ**: プライマリ: エンターテイメント / セカンダリ: 教育
- **コンテンツレーティング**: 4+
- **価格設定**: 無料

## 📋 Step 8: 申請・審査対応

### 8.1 申請提出

```bash
# App Store Connect で最終確認
open https://appstoreconnect.apple.com/apps/[app-id]/distribution
```

**提出前最終チェック:**
1. **ビルドが正常にアップロード済み**
2. **全ての必須項目が入力済み**
3. **スクリーンショット・アイコンが設定済み**
4. **プライバシー情報が設定済み**
5. **審査用メモが記入済み**

**申請提出:**
- **「審査に提出」ボタンをクリック**
- **確認画面で最終チェック**
- **「提出」をクリック**

### 8.2 審査ステータス確認

**ステータスの種類:**
```
「審査待ち」→「審査中」→「承認済み」or「却下」
```

**審査期間:** 通常1-3日、最大7日

### 8.3 却下時の対応

**よくある却下理由:**
1. **年齢制限不適切**: 内容と年齢制限の不一致
2. **プライバシー情報不足**: データ利用の説明不足
3. **機能不具合**: クラッシュやフリーズ
4. **ガイドライン違反**: App Storeガイドライン準拠不足

**対応手順:**
1. **却下理由の詳細確認**
2. **問題点の修正**
3. **新ビルドのアップロード**
4. **再申請の提出**
