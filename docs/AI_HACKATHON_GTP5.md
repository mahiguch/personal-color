# AIスタイリストで日常の自己表現をアップデート ─ 第3回 AI Agent Hackathon with Google Cloud 応募記事

## ハッカソン応募の背景
2025年8月5日から9月24日まで開催される「第3回 AI Agent Hackathon with Google Cloud」は、テーマに「AI Agentが、現実を豊かにする」を掲げ、Google Cloud の実行基盤と AI 技術を組み合わせた作品を求めています。審査では課題の新規性、解決策の有効性、実装品質と拡張性が重視され、提出物には GitHub リポジトリ、デプロイ環境、そして Zenn 記事が必須です。本記事はその提出条件を満たす応募用ブログとして、私たちが開発した「AIスタイリスト」がどのように現実世界の課題に挑むのかを共有します。

## プロジェクト概要：AIスタイリスト
AIスタイリストは、Google Cloud 上に構築したマルチエージェント型の美容・ファッション支援システムです。ユーザーがアップロードした写真をもとにパーソナルカラーを診断し、メイクやファッションの提案、学習コンテンツの生成までを一気通貫で支援します。プロダクションでは Cloud Run 上の FastAPI サーバーと Vertex AI（Gemini 2.5 Pro / Imagen 4.0）を連携させ、Flutter モバイルアプリ（iOS/Android）と Next.js ティザーサイトから利用できるようにしました。

## 対象ユーザーと課題、ソリューションの特徴（必須要件ⅰ）
- **背景・課題の整理**  
  - 診断へのアクセシビリティ: 専門診断は高額・予約負荷が高く、セルフ診断は主観に左右されがち。  
  - 活用ギャップ: 診断後の「具体的にどうすれば良いか」がわからず行動に移せない。  
  - 年齢適応: 世代によって適切なメイク・服装・解説の最適値が異なるが、従来サービスは一律。  
  - 毎日の意思決定コスト: 「今日の装い」を考える手続的な作業が累積して心理的負担になる。  

- **想定ユーザー（ペルソナ）**  
  - 自己表現に悩むZ世代（18〜24歳・学生〜社会人）  
    - 課題: 似合う色やスタイルがわからず自信が持てない。SNS/就活での第一印象を高めたいが、専門知識がない。  
    - ソリューション: 写真入力だけで Gemini が肌・瞳・髪色を解析し、季節タイプ + 推奨メイク手順を提示。Imagen 4.0 の Before/After で「変化の根拠」を視覚化。  
  - 忙しいワーキングパーソン（25〜39歳・共働き層）  
    - 課題: サロン診断に行く時間がない。TPO/季節/シーンに合わせた装いの判断が難しい。  
    - ソリューション: 「会議」「登壇」「デート」などシーンを宣言すると、最適な服装と色合わせを生成。カレンダー利用を前提に朝の意思決定を数十秒に短縮。  
  - 世代横断で美容を学びたいユーザー（40代以降・親子利用）  
    - 課題: 年齢・肌質に合わせたアドバイスが乏しく、親子で学べる教材も不足。  
    - ソリューション: 年齢推定と経験レベルで説明の語調と粒度を自動調整。ステップガイドと音声読み上げスクリプトで学習ハードルを下げる。  

- **ジョブストーリー（Job Stories）**  
  - 面接がある朝に、清潔感が伝わる配色で迷わないようにしたい。  
  - 新しい髪色に挑戦するとき、肌なじみと印象の差を事前に可視化したい。  
  - 子どもの発表会で、親子の写真が映える配色を手早く決めたい。  
  - 画面越しの会議でも顔が明るく見える色とメイクを数分で整えたい。  

- **解決アプローチと特徴**  
  - 客観診断: Gemini が肌・瞳・髪の特徴から季節タイプを判定し、根拠とともに提示。  
  - 視覚化: Imagen 4.0 による自然なメイク適用・コーデ生成で Before/After を即比較。  
  - 宣言型UI: 「やりたいこと（目的・シーン）」を宣言すると、最適解が自動生成（従来の検索→比較→取捨選択の手続きを代替）。  
  - 年齢適応: 推定年齢と経験レベルでカラー強度・解説の難易度を自動調整。  
  - 継続学習: ユーザーの保存・お気に入り・フィードバックをもとに、次回提案の精度と説明のパーソナライズを強化。  

- **ユースケース例**  
  - リモート会議前: 「顔が明るく見えるライトサマー寄りで、5分でできるメイク」  
  - 初対面の商談: 「信頼感を損なわず親しみが出るネイビー×コーラルの組合せ」  
  - 週末のおでかけ: 「春の屋外撮影で映える配色＋日焼け止めと相性の良い質感提案」  
  - 学校行事: 「親子でまとまりが出る配色・小物の差し色」  

- **導入効果（β版の観測指標）**  
  - 満足度 4.2/5.0、再利用意向 87%（クローズドβのアンケートより）  
  - 朝の準備での迷い時間を短縮（自由記述での最頻コメント: 「決めるのが楽になった」）  
  - Before/After の視覚化により家族・友人からのフィードバック獲得が容易に  

- **プロダクトが解く難しさ（差別化要因）**  
  - 単一AIでは難しい「診断→視覚化→実践」の一連体験を、Gemini×Imagenの協調で段差なく結合。  
  - 「手続き的操作から宣言的操作へ」の転換により、非専門ユーザーでも短時間で目的到達。  
  - 年齢適応と説明パーソナライズで、Z世代〜シニアまで違和感の少ない提案を実現。  

## システムアーキテクチャ（必須要件ⅱ）
### アーキテクチャ図
```mermaid
graph TD
    User[ユーザー\n(iOS/Android/Web)] --> App[Flutterモバイルアプリ]
    User --> Web[Next.jsティザーサイト]
    App -->|HTTPS API| API[FastAPI on Cloud Run]
    Web -->|診断要求| API
    API -->|解析要求| Gemini[Vertex AI Gemini 2.5 Pro]
    API -->|画像生成| Imagen[Vertex AI Imagen 4.0]
    API --> VirtualTryOn[Virtual Try-On Service\n(Imagen 4.0 Style Tuning)]
    API --> Storage[Cloud Storage\n生成画像・レポート]
    API --> Monitoring[Cloud Logging / Error Reporting]
    Storage --> CDN[Signed URL 配布]
```

### マルチエージェント構成
1. **診断エージェント**（Gemini 2.5 Pro）: 肌の色調、瞳のコントラスト、髪色を解析し Spring/Summer/Autumn/Winter の4タイプを判定。年齢推定結果から診断レポートの語調も調整します。  
2. **メイクアップ生成エージェント**（Imagen 4.0 + Gemini）: 元画像に対する自然なメイクの inpainting を行い、推奨コスメと塗布手順を生成。  
3. **ファッションコーディネートエージェント**（Imagen 4.0）: シーン別・季節別の全身コーディネート画像とカラー配色ガイドを提示。  
4. **ラーニングエージェント（開発中）**: Gemini による学習教材とアプリ内ガイダンスをパーソナライズするプランで、βテスター向けに段階的リリースを進めています。

## ユーザー体験フロー
1. **写真アップロード**: Flutter アプリまたはウェブから顔写真を送信。Firebase App Check と署名付きトークンでアクセスを制御。  
2. **並列エージェント実行**: Cloud Run 上の FastAPI が asyncio による非同期呼び出しで各 AI エージェントを制御し、最長でも25秒以内に結果を集約。  
3. **可視化とフィードバック**: Imagen 4.0 で生成した Before/After をサムネイル化し、Gemini がテキスト説明を返却。ユーザーはスタイルブックを保存し、次回の診断精度向上にフィードバックを送れます。  
4. **継続支援**: 診断ログと生成メタデータを整理し、BigQuery 連携とパーソナライズレコメンドの自動化を進めています。

## Google Cloud 活用と実装のポイント
- **Cloud Run**: FastAPI サーバーをコンテナ化し、トラフィック変動に合わせて自動スケール。リビジョンのカナリア切り替えと構造化ログで運用を安定化しました。  
- **Vertex AI Gemini / Imagen 4.0**: 画像解析と生成を組み合わせ、診断から視覚的提案までを一気通貫で提供。Gemini Flash へのフォールバックも用意し、API 障害時の継続稼働を確保。  
- **Cloud Storage + Signed URL**: 生成した画像・レポートを Cloud Storage に一時保管し、期限付き URL でアプリへ配信。プライバシーポリシーに沿って短期保管・自動削除を徹底しています。  
- **Firebase App Check**: Flutter アプリと Next.js ティザーサイトからの正規リクエストのみを許可し、不正アクセスを防止。  
- **Cloud Logging / Error Reporting**: 生成失敗時のトレースを即座に検知し、再実行とプロンプトチューニングサイクルを自動化。

## 審査観点へのアプローチ
- **課題の新規性**: 従来のパーソナルカラー診断が抱えていた「高額・主観的・年齢非対応」という課題を、AI エージェント協調で克服。親子やシニアを含む多世代対応でユニバーサルデザインを実現しています。  
- **解決策の有効性**: クローズドβ（2025年7月時点・参加ユーザー136名）で満足度 4.2/5.0、再利用意向 87% を確認。診断〜実践の平均所要時間を従来比 62% 削減しました。  
- **実装品質と拡張性**: Clean Architecture + DDD を採用し、各エージェントを疎結合化。Cloud Run と Terraform による IaC を整備し、メイク以外のカテゴリ（ヘア、ネイル）への展開も容易です。
