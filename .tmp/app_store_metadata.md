### アプリの説明

#### 短い説明（170文字以内）
```
小学5年生向けのパーソナルカラー診断アプリです。カメラで顔を撮影するだけで、AI技術を使ってイエベ・ブルベを判定し、似合う色を教えてくれます。楽しく学べるエンターテイメントアプリとして設計されています。
```

#### 詳細説明（4,000文字以内）
```
【パーソナルカラー診断アプリの特徴】

🎨 AI技術による高精度診断
最新のAI技術（Google Gemini）を活用し、顔写真から肌の色調を分析してパーソナルカラーを診断します。

📸 簡単な操作
1. アプリを起動
2. カメラで顔を撮影
3. 結果を確認するだけ！

小学5年生でも迷わず使える分かりやすいデザインです。

🌈 詳しい診断結果
・イエベ（イエローベース）・ブルベ（ブルーベース）の判定
・似合う色の具体的な提案
・分かりやすい解説

👶 安全・安心設計
・撮影した画像は診断後に自動削除
・個人情報の保存なし
・子供にも安心な設計

🎯 教育効果
色彩感覚を育み、自分に似合う色を知ることで、おしゃれや色選びの基礎知識が身につきます。

【こんな方におすすめ】
- 小学生のお子様

- パーソナルカラーを知りたいファミリー
- 楽しく学習したい親子

【技術仕様】
- iOS 14.0以降対応
- iPhone専用
- カメラ機能必須
- インターネット接続必須

プライバシーを重視し、安全に楽しく学べるエンターテイメントアプリです。
```

## 📸 スクリーンショット要件

### iPhone 6.7インチ（必須）
- 解像度: 1290 x 2796 pixels
- フォーマット: JPG or PNG
- 最大5枚

### 必要な画面キャプチャ
1. **ホーム画面**: アプリ起動時の魅力的な画面
2. **カメラ画面**: 撮影操作の様子
3. **診断結果画面**: イエベ・ブルベ結果表示
4. **色提案画面**: 似合う色の具体的な提案
5. **説明画面**: 分かりやすい解説表示

### スクリーンショット作成手順
```bash
# iOS Simulatorで以下の手順
1. Xcode → Open Developer Tool → Simulator
2. Device: iPhone 15 Pro Max (6.7インチ)
3. アプリを起動して各画面をキャプチャ
4. Device → Screenshots → Save to Files
```

## 🔒 プライバシー情報

### データの収集
**収集しない**

### カメラ使用目的
```
パーソナルカラー診断のために顔写真を撮影します。
撮影した画像は診断完了と同時に自動削除され、
デバイスやサーバーに保存されることはありません。
```

### データの取り扱い
- 撮影画像: 診断処理のみに使用、即座に削除
- 個人情報: 一切収集・保存しない
- 分析結果: デバイス上でのみ表示、保存なし
