# AI生成メイク画面改善 - 要件定義書

## 1. 概要

### 1.1 プロジェクト概要
AI生成メイク画面において、現在の単純な画像表示から、ユーザーがメイクの変化を理解し、実践的な知識を得られる包括的な体験に改善する。

### 1.2 改善対象
- 画面パス: `lib/features/makeup/presentation/pages/ai_makeup_recommendation_page.dart`
- 現状: AI生成画像のみを中央表示する最小限の実装
- 目標: Before/After比較、メイク解説、年齢適応型UI を備えた教育的価値の高い画面

### 1.3 改善の目的
- ユーザーの変化実感向上（Before/After比較）
- メイク教育・学習機能の提供
- 年齢に適したコンテンツ表示
- アプリ全体のエンゲージメント向上

## 2. 機能要件

### 2.1 優先順位付き機能リスト

#### 【最優先】Phase 1: ビフォー・アフター比較機能
**FR-001: Before/After並列表示**
- オリジナル画像とAI生成メイク画像を左右並列で表示
- 両画像のサイズ・位置を統一
- レスポンシブデザイン対応

**FR-002: 差分ハイライト機能**
- メイクが適用された箇所の視覚的強調表示
- 変化箇所への注意喚起
- ハイライト表示のON/OFF切り替え

#### 【次優先】Phase 2: メイク解説・ポイント説明
**FR-003: ステップバイステップ解説**
- メイク手順の段階的説明
- 各ステップの詳細説明テキスト
- 手順の順序表示（1. → 2. → 3.）

**FR-004: 年齢適応型コンテンツ**
- 撮影画像からの年齢推定機能
- 推定年齢に基づくメイク経験レベルの自動判定
- 年齢層に適した説明文・用語選択

**FR-005: パーソナルカラー理論説明**
- 年齢に応じた専門用語使用レベルの調整
- なぜその色が似合うかの理論的説明
- 実践的なアドバイス提供

#### 【将来機能】Phase 3: 動画・アニメーション対応
**FR-006: Veo3動画生成統合**
- メイク手順の動画生成機能
- Veo3 APIとの連携
- 動画の埋め込み表示

#### 【Phase 3】商品推薦機能 (将来実装)
**FR-007: パーソナルカラー×年齢別商品推薦**
- パーソナルカラー診断結果画面からの商品推薦起動
- 診断結果 + 撮影画像をAPIに送信して年齢・性別適応商品取得
- 年齢グループ別商品フィルタリング（child/student/adult）

**FR-008: 商品表示・購入連携**
- 商品カード形式での表示（画像・価格・ブランド名）
- Amazon購入リンクへの外部ブラウザ遷移
- 年齢適応された商品説明文表示

### 2.2 除外機能
以下の機能は実装対象外：
- 共有・保存機能
- 再生成機能
- SNS連携機能
- 商品詳細ページ・レビュー機能
- 高度なフィルタリング機能

### 3.2 ユーザビリティ要件

- **NF-011**: 文字列リソースの外部化

## 4. システム要件

### 4.1 技術制約
- **現在のAPI仕様**: 変更不可
- **新データフィールド**: 追加可能
- **画像処理場所**: サーバーサイド
- **オフライン対応**: 不要

### 4.2 データ要件
**新規追加が必要なデータフィールド（MakeupRecommendationエンティティ）:**
- `estimatedAge?: number` - 推定年齢
- `makeupExperienceLevel?: string` - メイク経験レベル
- `stepByStepInstructions?: List<MakeupStep>` - ステップ別手順
- `highlightAreas?: List<HighlightArea>` - 差分ハイライト領域
- `personalColorExplanation?: string` - パーソナルカラー理論説明
- `veo3VideoUrl?: string` - 生成動画URL（将来用）

**商品推薦機能用データフィールド（Phase 3）:**
- `MakeupProduct`エンティティ拡張:
  - `imageUrl?: string` - 商品画像URL
  - `amazonUrl?: string` - Amazon購入リンク
  - `colors: List<string>` - カラーバリエーション
  - `ageGroup?: string` - 対象年齢グループ
  - `difficultyLevel?: string` - 使用難易度

### 4.3 API要件
**必要な追加APIレスポンス:**

## 5. ユーザーストーリー

### 5.1 基本フロー
```
AS ユーザー
WHEN AI生成メイク画面を開くと
THEN オリジナル写真とAI生成画像が並んで表示される
AND メイクした箇所がハイライトされる
AND 年齢に適したメイク解説が表示される
```

### 5.2 年齢別体験フロー
```
GIVEN 推定年齢が10-15歳の場合
THEN 簡易な用語と基本的な手順で説明される
AND "ナチュラルメイク"中心の内容が提示される

GIVEN 推定年齢が20-30歳の場合
THEN 中級レベルの用語と詳細手順で説明される
AND より幅広いメイクテクニックが提示される
```

## 8. リスク・制約事項

### 8.1 技術的リスク
- **年齢推定精度**: 画像解析の精度に依存
- **API応答時間**: サーバーサイド処理による遅延
- **Veo3統合**: 将来機能の技術的実現可能性

### 8.2 UX リスク
- **情報過多**: 1画面表示による情報密度の問題
- **年齢適応**: 不適切な年齢判定によるUX悪化

### 8.3 運用制約
- **既存API制約**: 現仕様の変更不可
- **開発リソース**: 段階的実装が必要
