# パーソナルカラー診断テスト用画像

このディレクトリには、Geminiプロンプトのテスト用画像を配置します。

## 必要な画像ファイル

### 各パーソナルカラータイプのサンプル画像

1. **spring_sample.jpg** - スプリングタイプの人の顔写真
   - 特徴: 明るく華やかな印象、イエローベースの肌、明るい髪色
   - サイズ: 推奨 512x512px以上
   - 形式: JPEG

2. **summer_sample.jpg** - サマータイプの人の顔写真
   - 特徴: 上品で涼しげな印象、ブルーベースの肌、ソフトな髪色
   - サイズ: 推奨 512x512px以上
   - 形式: JPEG

3. **autumn_sample.jpg** - オータムタイプの人の顔写真
   - 特徴: 深みのある暖かい印象、イエローベースの肌、深い髪色
   - サイズ: 推奨 512x512px以上
   - 形式: JPEG

4. **winter_sample.jpg** - ウィンタータイプの人の顔写真
   - 特徴: はっきりと鮮やかな印象、ブルーベースの肌、コントラストの強い髪色
   - サイズ: 推奨 512x512px以上
   - 形式: JPEG

### エラーケーステスト用画像

5. **blurry_sample.jpg** - ぼやけた画像
   - 診断に不適切な画質をテストするため
   - 意図的にピンボケやブレがある画像

6. **no_face_sample.jpg** - 顔が写っていない画像
   - 風景、物、動物など人の顔以外の画像
   - 顔検出エラーのテスト用

## 画像準備のガイドライン

### 撮影条件
- **照明**: 自然光が理想的（窓際など）
- **背景**: シンプルで無地の背景
- **角度**: 正面から撮影
- **表情**: 自然な表情

### 画質要件
- **解像度**: 最低512x512px、推奨1024x1024px
- **ファイルサイズ**: 1MB以下に圧縮
- **形式**: JPEG形式
- **品質**: 圧縮品質85%以上

### プライバシー配慮
- テスト用画像は個人情報保護に配慮
- 使用許可を得た画像のみ使用
- テスト完了後は適切に削除
