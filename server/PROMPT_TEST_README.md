## 🎯 プロンプト設計の特徴

### 小学5年生向け配慮
- **平易な語彙**: 専門用語を避け、わかりやすい表現を使用
- **ポジティブな表現**: 励ましと肯定的なメッセージを重視
- **適切な長さ**: 50-150文字の説明で集中力を維持

### 診断精度向上
- **多角的分析**: 肌・髪・瞳の総合的評価
- **照明補正**: 撮影条件による色の歪み考慮
- **一貫性チェック**: 特徴間の矛盾検出

### エンターテイメント性
- **楽しい表現**: 診断を楽しい体験として提供
- **視覚的説明**: 色の名前と効果を分かりやすく説明
- **実用的アドバイス**: 具体的なファッション提案

## 🚀 実行方法

### 1. 事前準備
```bash

gcloud auth application-default login

# プロジェクト設定
gcloud config set project YOUR_PROJECT_ID

# Vertex AI API有効化
gcloud services enable aiplatform.googleapis.com
```
